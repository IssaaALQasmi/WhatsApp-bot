# بوت تذكير واتساب (WhatsApp Reminder Bot)

بوت واتساب باستخدام Node.js يقوم بتذكير المستخدمين بمهامهم عبر إرسال رسائل واتساب وإجراء اتصالات (رنّة) في الوقت الذي يحدده المستخدم.

## المميزات

- تحليل رسائل المستخدم لاستخراج محتوى التذكير والوقت باستخدام الذكاء الاصطناعي
- جدولة التذكيرات وحفظها في قاعدة بيانات بسيطة (ملف JSON)
- إرسال رسائل تذكير في الوقت المحدد
- إجراء اتصال واتساب (رنّة) كتنبيه إضافي
- إمكانية عرض قائمة التذكيرات المجدولة

## متطلبات النظام

- Node.js (الإصدار 14 أو أحدث)
- اتصال إنترنت مستقر
- هاتف ذكي به تطبيق واتساب لمسح رمز QR عند بدء التشغيل

## التثبيت

1. قم بتثبيت Node.js من الموقع الرسمي: https://nodejs.org/

2. قم بتنزيل أو استنساخ المشروع:
```
git clone https://github.com/your-username/whatsapp-reminder-bot.git
cd whatsapp-reminder-bot
```

3. قم بتثبيت المكتبات المطلوبة:
```
npm install
```

4. قم بتعديل ملف `.env` وإضافة مفتاح API للذكاء الاصطناعي:
```
AI_API_KEY=your_api_key_here
DB_FILE_PATH=./reminders.json
TIMEZONE=Asia/Riyadh
```

## كيفية الاستخدام

1. قم بتشغيل البوت:
```
npm start
```

2. عندما يظهر رمز QR، قم بمسحه باستخدام تطبيق واتساب على هاتفك (من قائمة الإعدادات > الأجهزة المرتبطة > الربط بجهاز).

3. بمجرد نجاح الاتصال، يمكن للمستخدمين التفاعل مع البوت بالطرق التالية:

   - **إضافة تذكير جديد**:
     أرسل رسالة تبدأ بـ "ذكرني" أو "تذكير" متبوعة بمحتوى التذكير والوقت.
     مثال: "ذكرني بموعد الطبيب غدًا الساعة 3 مساءً"

   - **عرض التذكيرات المجدولة**:
     أرسل "تذكيراتي" أو "عرض التذكيرات"

## هيكل المشروع

- `bot.js` - الملف الرئيسي للبوت، يربط مكتبة الواتساب بباقي المكونات
- `reminderService.js` - خدمة إدارة وجدولة التذكيرات
- `aiParser.js` - تحليل رسائل المستخدم باستخدام الذكاء الاصطناعي
- `database.js` - إدارة تخزين التذكيرات في ملف JSON
- `.env` - ملف المتغيرات البيئية

## ملاحظات هامة

- لاستخدام خدمة الاتصال (الرنّة)، يجب أن تدعم المكتبة المستخدمة هذه الخاصية. في البوت الحالي، تم إضافة بديل (رسالة تنبيه) في حالة عدم توفر الدعم.
- البوت يحتفظ بصلاحية الدخول في المجلد `.wwebjs_auth` الذي يتم إنشاؤه تلقائيًا. لا تحذف هذا المجلد لتجنب الحاجة لمسح رمز QR في كل مرة.
- تأكد من أن الهاتف الذي يشغل البوت متصل بالإنترنت دائمًا لضمان إرسال التذكيرات في وقتها.

## المشاكل الشائعة

- **مشكلة الاتصال**: إذا فشل البوت في الاتصال، تأكد من استقرار اتصال الإنترنت ومن أن الهاتف المرتبط بالبوت متصل بالإنترنت أيضاً.
- **مشكلة تحليل الوقت**: إذا كان البوت لا يفهم صيغة الوقت بشكل صحيح، حاول استخدام صيغ أكثر وضوحاً مثل "اليوم الساعة 5 مساءً" أو "غدا في 3:00".
